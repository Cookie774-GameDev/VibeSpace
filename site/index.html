Main feature · AI calling

Phone your AI like a friend. It picks up.

VibeSpace ships a real calling layer. Dial your assistant from any phone through Twilio, or tap the green Call button in the desktop app for a free LiveKit call. Jarvis voice can talk, search, summarize files, create tasks, and call you when something important happens.

i

Call from any phone

Path A uses Twilio Programmable Voice + Media Streams so a real number can route into Sage, your voice Jarvis.

ii

Tap to call in-app

Path C uses LiveKit WebRTC. No telco minutes, lower latency, and the same assistant brain inside your VibeSpace window.

iii

Let Sage call you

Outbound triggers can ring your phone for build failures, deadline alerts, errors, or manual check-ins.

The 9pm dog walk

Download

Three steps. Any desktop.

VibeSpace supports Windows, macOS, and Linux. Use the one-liner, approve the installer, then launch VibeSpace and add your first provider key.

Windows 10/11

irm https://raw.githubusercontent.com/Cookie774-GameDev/VibeSpace/main/install/install.ps1 | iex
  1. Open PowerShell.
  2. Run the one-line installer.
  3. Type Jarvis in a terminal or launch VibeSpace from Start Menu.

macOS 12+

curl -fsSL https://raw.githubusercontent.com/Cookie774-GameDev/VibeSpace/main/install/install.sh | bash
  1. Open Terminal.
  2. Run the curl installer.
  3. Open VibeSpace from Applications.

Linux

curl -fsSL https://raw.githubusercontent.com/Cookie774-GameDev/VibeSpace/main/install/install.sh | bash
  1. Open your terminal.
  2. Run the installer; deb/rpm/AppImage auto-detects.
  3. Launch VibeSpace from your app menu.

FAQ

Questions before you install.

Is VibeSpace only for coding?

No. Coding is the sharpest use-case, but VibeSpace also handles research, calls, tasks, memory, skills, agents, notes, actions, and project context.

What models work today?

Anthropic, OpenAI, Google Gemini, Groq, and Ollama are live adapters. Other providers already have UI/key persistence and can be promoted as adapters ship.

Is calling fully done?

The backend, bridge, settings, in-app LiveKit path, Twilio endpoints, PIN flow, audit logging, and outbound API are in place. Some hardening remains: real Cartesia voice IDs, verbal-confirm enforcement, audit viewer UI, trigger callsites, and provisioning automation.

Can it run offline?

Yes. Offline mode routes chat to Ollama. Local model support is live and keyless.

What is the official website?

The official home is vibespaceos.com. Downloads and one-line installers always resolve through the GitHub releases page, so both stay in sync.
